# 홈페이지로 [이동하기](https://cet4713.cafe24.com/) 
### * 스크롤을 통해 가로로 움직일 수 있습니다! *

### 기획 의도
게임을 하다보면 다양한 아이템들을 획득한다. 그 중 아이템을 최소 며칠에서 최대 몇 개월까지 모으는 상태가 존재하게 된다.
유저들의 편의성과 궁금증을 조금이라도 해소하고자 내가 얻고싶은 아이템이 몇 주가 필요한지 계산해주는 사이트가 있었으면 해서 만들게 되었다.
